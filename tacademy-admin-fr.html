Plateforme de cours en ligne TAcademy (administration)

L’objectif de ce site est de distribuer l’accès à un catalogue de leçons vidéo créées par le responsable du produit. Le client souhaitait une plateforme permettant de partager efficacement son matériel pédagogique, dans un esprit similaire à Skillshare. Le site prend en charge le paiement en ligne et comprend une base de données contenant les informations des utilisateurs. Un serveur API est également dédié aux fonctionnalités principales. La plateforme présente trois perspectives : l’administrateur, les utilisateurs inscrits et les visiteurs non inscrits.
